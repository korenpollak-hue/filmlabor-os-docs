6 Regeln

Nach Jake Van Clief. So arbeitet der AI-Agent.

0

Memory zuerst

Was weiss ich schon? Memory-Datenbank pruefen.

1

Skills zuerst

Skill existiert? Folge ihm. Keiner da? Baue einen.

2

CONTEXT.md zuerst

Lies sie bevor du in einem Ordner arbeitest.

3

Scripts wiederverwenden

scripts/ pruefen bevor du neuen Code schreibst.

4

Docs ueber Outputs

context/ und SKILL.md sind Wahrheit.

5

USER / SYSTEM Trennung

context/ + workspace/ = User. code/ + scripts/ = System.
